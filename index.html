To make this work, you need to update your Google Apps Script to handle CORS properly. Add this line to your handleRequest function:

return ContentService.createTextOutput(JSON.stringify(result)).setMimeType(ContentService.MimeType.JSON).setHeaders({'Access-Control-Allow-Origin': '*'});
